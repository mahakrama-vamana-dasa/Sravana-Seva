## 📖 Overview

This repository is not merely a collection of notes. It is a **structured knowledge system** designed to organize, connect, and internalize Gauḍīya Vaiṣṇava philosophy across lectures, scriptures, and practical applications.

It integrates multiple knowledge layers—lectures, concepts, analogies, songs, and letters—into a unified, interconnected system to support the journey from *śravaṇa* (hearing) to *vijñāna* (realized knowledge).

---

## 🏛️ Knowledge Layers

The vault is organized into the following interconnected layers:

1. **Lecture Notes (Primary Input)**
Source material captured from live classes and systematic study.
2. **Concept System (Tattva Framework)**
The central organizing system where all knowledge is structured through tattva-based concepts and interconnected across sources.
3. **Verse Hubs (Śāstric Anchors)**
Verse-centric hubs that connect multiple lectures, realizations, and insights around specific śāstric references (*Bhagavad-gītā*, *Śrīmad-Bhāgavatam*, *Upaniṣads*, etc.).
4. **Analogies (Illustration Layer)**
Practical examples and metaphors used to simplify, teach, and internalize philosophical concepts.
5. **Songs & Bhajans (Meditative Layer)**
Integration of philosophy through the devotional compositions of Gauḍīya Ācāryas.
6. **Letters (Application Layer)**
Practical instructions guiding behavior, service, and daily conduct.

This vault follows a hybrid approach:
